Known Issues

Current Known Issues

  • Clicking Too Fast on Expander Icons: Rapid clicks on expander icons (e.g., in the build area) can cause them to get out of sync, showing incorrect states.
  • Opening a Save in a New Window/Tab: Saved force lists sometimes load incompletely in new tabs or windows, though refreshing the page often resolves it. Monitoring for frequency and checking for lists that fail to load fully even after refresh.
  • Faction Rules Adding Special Rules: Faction special rules that grant additional special rules (e.g., “All units in this Force gain the Poorly Equipped and War Cry Special Rules”) don’t add those rules to the PDF summary or apply them to units/components directly.

Known Issues with Workarounds

These issues can be mitigated using custom components until a better fix is implemented:

  • Missing Unit Options: Need to add bow/musket options for FotF Native commanders.
  • Climbing Gear Cost with Explosives: Adding Climbing Gear to a unit with Explosives access incorrectly adds +2 points instead of being free. Workaround: Add a custom component with a -2 point cost to offset this. Status: On hold pending higher-priority fixes.

Not Yet Implemented

These features are planned but not yet available—they are being tracked for future updates:

  • Individualized selection of explosives (e.g., per-model choice of Grenades vs. Firepots).
  • Add years active and/or sourcebook info to the faction selector for historical context.
  • Option to show/hide Climbing Gear and Torch unit options in the UI.
  • Force-reset button to clear the current list and start fresh.
  • Default build-side interior expanders to open for easier access.
  • Password-change capability for user accounts.
